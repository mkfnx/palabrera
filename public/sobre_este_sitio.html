¿Por qué se creó este sitio?

Este sitio se creó como un ejercicio de aprendizaje en Ingeniería y Visualización de Datos.

Se explora un conjunto de datos de interés general y se intenta obtener y presentar información útil a partir de ello, en este caso, visualizar fácilmente los temas que se trataron en las conferencias matutinas del presidente a partir de las palabras mencionadas.

¿Cómo se recopilaron los datos?

El contenido de las conferencias se recopiló de las transcripciones publicadas en el sitio de gobierno del presidente de México: https://presidente.gob.mx/secciones/version-estenografica/

¿Cómo se analizan los datos?

Se usó la biblioteca WordCloud for Python, la cual realiza técnicas sencillas de procesamiento de textos para obtener un diccionario o mapa de frecuencias de las palabras más mencionadas en un texto, en este caso, la transcripción de las mañaneras y a partir de ello crea un gráfico de nube de palabras.

La gráfica de barras se crea usando el mismo mapa de frecuencias creado por WordCloud for Python. Esta gráfica se incluye en conjunto con la nube de palabras debido a que, aunque es visualmente menos atractiva que la nube de palabras, indica exactamente la frecuencia de cada palabra y las muestra en orden de las más mencionadas.

Palabras Omitidas

Cómo parte del procesamiento de datos se omiten algunas palabras que no aportan mucha información al análisis. Estas palabras se conocen como stopwords.

Para este análisis se consideraron dos listas de stopwords. La primera es una lista estándar de stopwords del idioma español, la cual fue extraída de la biblioteca NLTK. La segunda lista es de elaboración propia y contiene palabras comúnmente mencionadas en las mañaneras.
